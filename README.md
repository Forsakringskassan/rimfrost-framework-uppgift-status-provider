# rimfrost-framework-uppgift-status-provider

Provides a default provider implementation for the rimfrost-framework-uppgift-status-provider-interface
interface that returns hardcoded ids for the different uppgift statuses defined in the interface.

The current hardcoded status-to-id mapping consists of:

- Planerad -> c7477256-eabd-4ff2-833a-2db9051caf86
- Tilldelad -> 0b202a3f-052d-450a-8866-ae141a80d289
- Avslutad -> 5161d678-3835-454c-979b-dc1c9e06beac
